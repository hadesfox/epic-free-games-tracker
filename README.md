# Epic Games Store 免费游戏记录

自 2018 年 12 月首次赠送 Subnautica 以来，Epic Games Store 已赠送了数百款免费游戏。这个页面完整记录了每一次赠送。

## 功能

- **603 条赠送记录**，覆盖 2018 至今
- **Steam 官方海报**（519/603 覆盖，86%）
- **中英文双语名称**显示
- **卡片 / 列表**两种布局切换
- **一键刷新**：点击刷新按钮自动从 Epic 官方 API 获取最新赠送记录
- 按年份筛选、名称搜索、多种排序
- 纯静态 HTML，无需服务器，浏览器直接打开即可使用

## 使用方式

1. 下载 `epic-free-games.html`
2. 用浏览器直接打开
3. 点击右上角「刷新数据」按钮获取最新记录
